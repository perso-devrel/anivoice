# KoeDub Roadmap

프로젝트의 향후 개발 방향입니다. 우선순위는 변경될 수 있습니다.
기능 제안은 [GitHub Discussions](https://github.com/perso-devrel/KoeDub/discussions)에서 환영합니다.

---

## v1.1 — 사용자 경험 개선

- [ ] 더빙 결과 미리보기 플레이어 개선 (구간 반복, 속도 조절)
- [ ] 대시보드 프로젝트 일괄 관리 (선택 삭제, 일괄 다운로드)
- [ ] 라이브러리 검색 고도화 (언어, 태그, 인기순 필터)
- [ ] 모바일 UI 최적화 (터치 제스처, 반응형 개선)
- [ ] 다크/라이트 테마 전환

## v1.2 — 기능 확장

- [ ] 실시간 더빙 진행 알림 (이메일 / 푸시)
- [ ] 더빙 이력 비교 (이전 버전과 diff)
- [ ] 사용자 간 공동 작업 (프로젝트 공유)
- [ ] 추가 더빙 언어 지원
- [ ] 자막 SRT/VTT 내보내기 형식 선택

## v1.3 — 결제 및 수익화

- [ ] Stripe 결제 연동 (크레딧 구매)
- [ ] 구독 플랜 (월간/연간)
- [ ] 크레딧 선물 기능
- [ ] 사용량 분석 대시보드

## v2.0 — 플랫폼 확장

- [ ] React Native 모바일 앱 (Expo)
- [ ] REST API 공개 (외부 연동)
- [ ] Docker 컨테이너 배포 지원
- [ ] 셀프 호스팅 가이드
- [ ] 플러그인 시스템 (커스텀 TTS 엔진)
